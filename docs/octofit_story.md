
# Mergington High School을 위한 GitHub Copilot Agent Mode 기반 피트니스 앱 구축  
## Mergington High School을 위한 OctoFit Tracker 애플리케이션 스토리  

Paul Octo는 Mergington High School에서 8년 이상 근무해 온 체육 교사입니다. 체육 수업에 대한 그의 열정과 창의적인 접근 방식에도 불구하고, 그는 학생들이 학교를 벗어난 이후 신체 활동이 점점 줄어들고 있다는 점을 점점 더 걱정하게 되었습니다. 많은 학생들이 필수 체육 수업 외에는 거의 운동을 하지 않는다고 털어놓았습니다.  

“체육 수업에서의 기술 통합(Technology Integration in Physical Education)”이라는 주제의 전문성 개발 컨퍼런스에 참석한 후, Paul은 해결책을 만들어야겠다는 영감을 받았습니다. 그가 원했던 것은 다음과 같은 조건을 충족하는 것이었습니다.  

1. 피트니스 트래킹을 재미있고 몰입감 있게 만들 것  
2. 친근한 경쟁을 통해 긍정적인 또래 압력을 형성할 것  
3. 학생들의 진행 상황을 원격으로 모니터링할 수 있을 것  
4. 개인별 체력 수준에 기반한 맞춤형 가이드를 제공할 것  

## OctoFit Tracker의 탄생  

Paul은 점심시간마다 메모장에 자신의 아이디어를 간단히 스케치했습니다. 그는 학생들이 운동 기록을 남기고, 성취 배지를 획득하며, 매달 진행되는 피트니스 챌린지에서 경쟁할 수 있는 앱을 구상했습니다. 하지만 기본적인 코딩 지식만 가진 체육 교사였던 Paul에게 기술적인 부분은 큰 장벽처럼 느껴졌습니다.  

그때 그는 Mergington High School IT 부서의 책임자인 Jessica Cat에게 도움을 요청했습니다. Jessica는 우리 저장소의 지침과 프롬프트를 사용하는 것을 추천했습니다.  

### 기술 기획 단계  

개발을 시작하기 전에 Paul과 Jessica는 OctoFit Tracker의 지침과 프롬프트를 꼼꼼히 검토했습니다. 이를 통해 기술 표준을 준수하고 검증된 디자인 패턴을 활용할 수 있는 탄탄한 기반을 마련할 수 있었습니다.  

Paul과 IT 팀은 함께 OctoFit Tracker의 핵심 요구사항을 다음과 같이 정의했습니다.  

### 사용자 경험 목표  

- 10대 학생들을 위해 특별히 설계된 단순하고 직관적인 인터페이스  
- 마찰을 최소화하는 빠른 활동 기록  
- 학생 개인정보를 존중하는 소셜 기능  
- 지속적인 참여를 유도하는 게임화 요소  

## 현재 개발 진행 상황  

Paul과 Jessica는 GitHub Codespaces 환경을 설정했고, GitHub Copilot agent mode를 활용해 놀라운 속도로 개발을 진행하고 있습니다. OctoFit Tracker 프로토타입에는 다음 기능이 포함될 예정입니다.  

- 동작하는 사용자 등록 시스템  
- 달리기, 걷기, 근력 운동에 대한 기본 활동 기록  
- 팀 대항 경쟁을 위한 초기 프레임워크  
- 학생 진행 상황을 보여주는 간단한 대시보드  

## Paul의 다음 단계  

기본 인프라가 갖춰진 지금, Paul은 다음과 같은 작업에 집중하고 있습니다.  

1. 서로 다른 유형의 활동을 공정하게 비교할 수 있는 흥미로운 포인트 시스템 개발  
2. 다양한 학생 관심사를 반영한 동기 부여 챌린지 설계  
3. 과도하지 않으면서도 꾸준함을 유도하는 알림 시스템 구축  
4. 추가적인 지원이나 동기 부여가 필요한 학생을 식별하는 데 도움이 되는 리포트 디자인  

IT 부서는 GitHub Copilot agent mode가 개발 속도를 크게 높여 주었고, 그 덕분에 Paul이 기술 구현보다는 교육적인 측면에 더 집중할 수 있었다는 점에 깊은 인상을 받았습니다. Jessica Cat 역시 OctoFit Tracker가 커스텀 지침과 프롬프트 파일을 효과적으로 활용하고 있다는 점에 특히 만족하고 있습니다.  

### 워크숍 개요  

이 워크숍에서는 다음 내용을 다룹니다.  

1. **GitHub Codespaces**를 사용한 개발 환경 설정  
2. **GitHub Copilot**을 활용해 다양한 기술 스택 전반에서 개발 가속화  
3. Copilot agent mode의 도움을 받아 **OctoFit Tracker** 앱의 핵심 구성 요소 구축  
4. **GitHub Copilot agent mode**와 함께 작업할 때의 베스트 프랙티스 및 프롬프트 작성 기법 학습  

### 애플리케이션 기능  

**OctoFit Tracker**에는 다음 기능이 포함됩니다.  

- 사용자 프로필  
- 활동 기록 및 추적  
- 팀 생성 및 관리  
- 경쟁형 리더보드  
- 개인 맞춤형 운동 제안

### OctoFit Tracker 피트니스 애플리케이션 기술 스택  

다음과 같은 최신 웹 애플리케이션 스택을 사용합니다.  

- **Frontend**: React.js  
- **Backend**: Python + Django REST API Framework  
- **Database**: MongoDB  
- **Development Environment**: GitHub Codespaces
